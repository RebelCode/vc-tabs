# Tabs Component

## Usage
```js
<tabs :options="tabsConfig" v-model="activeTab">
    <tab title="First Tab">
        First Tab Content
    </tab>
    <tab title="Second Tab">
        Secont Tab Content
    </tab>
    <tab title="Third Tab">
        Third Tab Content
    </tab>
</tabs>
```

## Development
Run `npm install` to install all dev dependencies.

Here is available npm commands.

Build library while developing
```sh
npm run dev
```

Build library for production
```sh
npm run production
```

Run e2e tests
```sh
npm run e2e
```
